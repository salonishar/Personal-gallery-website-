# Personal Gallery (CSS Grid Project)

A multi-page image gallery website built using **HTML and CSS Grid**.

This project displays different categories of images (Nature, City, Tech, Music, etc.) on the homepage.  
Each category opens a separate gallery page containing related images.

The main goal of this project was to practice:
- CSS Grid layout
- multi-page website structure
- image handling in HTML
- hover effects
- folder organization
- navigation using anchor tags

---

## Features

- Custom CSS Grid layout
- Clickable category cards
- Multiple gallery pages
- Hover effects on images
- Responsive image fitting using `object-fit`
- Organized folder structure
- Clean dark theme UI
